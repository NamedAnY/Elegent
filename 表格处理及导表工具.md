# 本地配置表信息读取与应用

<div class="section">

该流程用于在 **代码中读取并应用本地配置表**，确保数据在运行时能够正确被访问和使用。  

</div>

## 基本流程
<div class="section">

### 1. 表格创建
在 **cfg_c** 文件夹中创建表格，并填充数据。  
   - 建议保持 **表格文件名** 与 **sheet 名** 一致。  
   - 在表格的备注一栏写上 **bz**，否则代码读取时可能会出问题。  
   - 生成的表格数据可在 `game/bin/json` 中查看。  

---

### 2. 执行 Python 脚本
使用 Python 运行 `__run__.py` 文件。  
   - 前提：本地已安装 **Python 环境** 及第三方库 **openpyxl**。  

---

### 3. 代码中读取表格数据
- 在 `CfgDefine` 中定义配置类，继承自 `fx.BaseData`：  
  ```ts
  class MagicBoxCfg extends fx.BaseData {}
- 在 CfgManager 中创建对应的数据配置对象字段，并且在构造函数中进行实例化

---

### 4. 使用
完成以上步骤后，即可在代码中访问对应的配置对象，并使用其 get() 方法 等接口读取具体的数据内容。
